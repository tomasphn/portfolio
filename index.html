Response Repo

Ruby
Ruby on Rails
GPT-3 API
PSQL
HTML & CSS
Tailwind
Heroku

A Ruby on Rails web application that leverages the power of OpenAI's GTP-3 AI through the "ruby-openai" gem to generate custom responses to user-submitted phrases. The application utilizes an SQL database, managed by PSQL and Rails' Active Migration systems, to store responses and prompts for future access. To enhance user experience, the application features a user login system, allowing users to add their own responses and react to existing ones.

This personal project, developed in collaboration with a friend, allowed me to explore integration of AI technology into web applications and set the ground for our current project that uses GPT3 to analyze medical records.

A CLI gem that simplifies the process of building conventional file structures for different types of Ruby projects. Currently, it supports Sinatra projects and two file structure options, with plans to expand its capabilities to other frameworks and options in the future.

The goal of developing and publishing this gem was to create an application I could make practical use of while also gaining experience adhering to conventional file and code organization practices. I utilized my understanding of Ruby OOP from my studies at Launch School to construct a custom class for easy organization and creation of file trees.
